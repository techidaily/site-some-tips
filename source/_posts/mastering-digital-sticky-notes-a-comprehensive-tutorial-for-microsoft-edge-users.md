## Mastering Digital Sticky Notes: A Comprehensive Tutorial for Microsoft Edge Users

Windows 10 recommended you a new browser Microsoft Edge. Microsoft Edge allows you to make notes on the webpage. You can  write, doodle, draw, or highlight on webpages. In this article you will learn how to make a web note in Microsoft Edge.   
  
**First, open Microsoft Edge.**   
  
 Usually it is pinned on the bottom task bar.
